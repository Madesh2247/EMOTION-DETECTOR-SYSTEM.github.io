## 📖 About

The Emotion Detector System is an AI-powered computer vision project designed to recognize human facial emotions in real time.

Using a Raspberry Pi with a Pi Camera, the system captures facial images, detects faces with OpenCV, and classifies emotions using a pre-trained Facial Emotion Recognition (FER) model.

This project demonstrates how embedded systems and AI can be combined to create lightweight intelligent applications.

---

## ✨ Features

- 🎥 Real-time facial emotion detection
- 📷 Pi Camera integration
- 🧠 Facial Emotion Recognition (FER)
- 😀 Detects emotions such as Happy, Sad, Angry, Neutral, Fear and Surprise
- ⚡ Lightweight Raspberry Pi deployment
- 🖥 Python-based implementation
- 📊 Real-time prediction workflow
